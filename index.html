About

I'm a passionate front-end developer who thrives on crafting intuitive and scalable web experiences. Over the years, I’ve honed my skills in HTML, CSS, JavaScript, and TypeScript, building responsive and dynamic interfaces using frameworks like React, Angular, and Next.js. For me, clean code and thoughtful UI go hand-in-hand.

At Sumeru Software Solutions, I’ve led modules from concept to deployment—whether it’s creating recursive UI flows, integrating global payment systems, or tailoring Keycloak for seamless authentication. I take pride in turning complex Figma designs into pixel-perfect, production-ready components and mentoring interns along the way.

Beyond the code, I love solving real-world problems with tech—be it setting up Supabase backends or building reusable libraries that boost development speed. Every line of code I write is driven by a desire to simplify the user journey and bring design to life.
